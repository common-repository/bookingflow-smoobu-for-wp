== Description ==

#### Free Wordpress Calendar Plug-in integrated with Smoobu`s API 

Bookingflow offers you an amazing Wordpress calendar Plugin integrated with Smoobu. Our calendar is fully customizable and offers you almost unlimited options to meet your design needs. You can select from numerous colors for the all areas of your calendar and you can show up to 9 months on one page. 

This Plugin comes with a very easy to use Wordpress Dashboard. Once set up and installed into your Wordpress site you don’t have to maintain it again - Smoobu will do the work for you.

== Installation ==

To install this Plugin please follow:

* Please install the „Smoobu Calendar Plugin“ in your Wordpress Dashboard under Plugins
* Please enter the Smoobu API Key to the plugin - you will find it under Smoobu > Settings > for developers
* Please copy the Webhook URL which you can find in the plugin - enter it under Smoobu > Settings > for developers
* Your calendar shall now synchronize automatically - if not please press once the „synchronize“ button

#### Have fun and enjoy your beautiful calendar.


== Frequently Asked Questions ==

= How to get started? =

1. Download and install the plugin.
2. Go to **Smoobu Calendar** -> **General Settings** in the left menu.
3. Fill your **API Key** and click **Save Changes**
4. You now can add the calendar as a widget, Gutenberg block or a shortcode.

= Where can I find a shortcode for my properties? =

You can find all your properties and shortcodes in **Smoobu Calendar** -> **My Properties** in the left menu.

= I have a question that is not answered! =

You can access an extended FAQ section upon installation. It will be available in **Smoobu Calendar** -> **FAQ** in the left menu. If you still can't find an answer to your question, feel free to contact us.
